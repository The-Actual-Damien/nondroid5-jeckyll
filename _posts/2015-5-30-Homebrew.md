---

MacにHomebrewでインストールしておくべきものをリストアップしてみた

grepやackよりもagでは！

```
$ brew install the_silver_searcher
```

ターミナルのスクリーン分割tmux

```
$ brew install tmux
```

diffを色付きで表示させる colordiff

```
$ brew install colordiff
```

.zshrcにdiffをcolordiffにエイリアスすると便利

```
$ alias diff='colordiff'
```

cmake

```
$ brew install cmake
```
