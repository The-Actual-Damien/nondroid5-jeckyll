## Live CDについて

###DVD rw のフォーマット

DVD rw を、完全に、フォーマットしたい時

```
$ sudo apt-get dvd+rw-tools
```

で、ツールをゲットし

```
$ dvd+rw-format -force /dev/sr0
```

と、-force オプションをつけた方が良いらしい
